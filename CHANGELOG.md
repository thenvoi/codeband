# Changelog

All notable user-facing changes to Codeband should be documented here.

This project is currently in alpha. Until the first stable release, minor versions may include breaking changes; migration notes should be linked from the relevant release entry.

## 0.1.1 - 2026-05-07

- Added OSS community files, issue templates, and CI workflow.
- Restructured the README into a shorter landing guide.
- Added focused authentication, configuration, and troubleshooting docs.
- Added PyPI publishing workflow (tag-driven, Trusted Publishing under the Band PyPI org) and `RELEASING.md`.
- Changed agent peer discovery to rely on canonical platform descriptions with stable `role=...` and `framework=...` tokens, supporting lazy room invites across multiple agent instances.
- Breaking: removed user-configurable pool `description` fields from `codeband.yaml`; agent descriptions are now generated by Codeband so discovery prompts can rely on consistent role/framework metadata.
